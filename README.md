# -Expense-Tracker-Web-Application
The Expense Tracker is a web-based application help user manage their daily income and expenses .It allows users to add transactions, view income and expense summaries, and automatically calculates the available balance. The application uses browser Local Storage to save data, ensuring that transaction records remain even after refreshing the page.
